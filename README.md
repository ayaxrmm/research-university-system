# Information System of a Research-Oriented University

A Java-based university management system developed as part of the **Object-Oriented Programming (OOP)** course at **Kazakh-British Technical University (KBTU)**.

This project demonstrates the complete software development lifecycle, including requirements analysis, UML modeling, object-oriented design, and implementation of a role-based university management system.


## Project Overview

The goal of this project was to design and develop an information system for a research-oriented university that supports academic, administrative, and financial processes.

The system provides different functionality depending on the user's role and follows object-oriented design principles to ensure scalability and maintainability.


## Features

* User authentication
* Role-based access control
* Course registration
* Academic schedule management
* Student journal
* Attendance tracking
* Grade management
* Financial cabinet
* Transaction management


## User Roles

| Role       | Responsibilities                                                       |
| ---------- | ---------------------------------------------------------------------- |
| Student    | Register for courses, view schedule, transcript, and academic progress |
| Teacher    | Manage grades and attendance                                           |
| Admin      | Manage courses and student schedules                                   |
| Accountant | Track payments, manage transactions, generate financial reports        |
